{{ _("About the Data for London Library") }}

Share, find and use city data to improve lives for Londoners. The Data for London Library makes crucial datasets simpler and easier to find for researchers, service providers and innovators. With better access to data, you can help improve services, address challenges like housing and journey delays, and spark new ideas to help Londoners. The Data for London Library is part of the Data for London programme. See how Data for London can support your work.
